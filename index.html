UFlash Sprint 1 (Website)

Needs HTTPS (or localhost) for camera access. Torch control via web works mainly on some Android Chrome; iOS often won’t support torch, so sender falls back to screen flashing.

Mode

Open this page on two phones: one sender, one receiver.

Sender

PREAMBLE: 10101010
POSTAMBLE: 00000000

Status: idle

If torch isn’t supported, it will automatically use full-screen flashing (white/black).

Receiver

Status: idle

Luma: - | Bits buffered: 0

Bits(last 40): -
